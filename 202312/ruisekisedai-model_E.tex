In this study, we construct cumulative generations in order to capture cumulative changes. First, we divide the period from 1946 to 2020 into 15 non‑overlapping five‑year intervals and combine them cumulatively. This is referred to as the cumulative‑generation model. Figure \ref{fig:gr2} shows, for each interval before the combination, the number of registered papers, the number of papers that contain references, the number of cited papers, the number of citations, and the number of journals. All of these, except for the number of journals, exhibit exponential growth.
